Research Hub - دليل التشغيل والرفع
=====================================

══════════════════════════════════════
  تشغيل على جهازك (Windows)
══════════════════════════════════════

المتطلب الوحيد: Node.js
  حمّله من: https://nodejs.org  (اختر LTS)

خطوات التشغيل:
  1. افتح مجلد المشروع
  2. اضغط دبل كليك على START.bat
  3. المتصفح هيفتح تلقائياً على http://localhost:3000


══════════════════════════════════════
  رفع على الإنترنت مجاناً - Render
══════════════════════════════════════

  1. اعمل حساب على github.com
  2. ارفع كل ملفات المجلد على GitHub repository جديد
  3. اعمل حساب على render.com
  4. اضغط New ثم Web Service
  5. اربط الـ GitHub repository
  6. اضبط:
       Start Command: node server.js
  7. اضغط Deploy
  8. هتاخد رابط مجاني:
       https://research-hub-xxxx.onrender.com
